# 🖐️ AI Virtual Mouse

### Control Your Computer with Hand Gestures

An innovative **AI-based Virtual Mouse** that allows users to control their system cursor using **hand gestures** through a webcam. This project leverages **Computer Vision** and **Machine Learning** to replace traditional mouse interaction.

---


## 📌 Features

✨ Cursor Movement using Index Finger
✨ Left Click Detection
✨ Right Click Detection
✨ Drag & Drop Support
✨ Scroll Functionality
✨ Real-time Hand Tracking
✨ Smooth & Accurate Motion Control

---

## 🧠 How It Works

1. Webcam captures live video feed
2. Hand is detected using ML-based tracking
3. Finger landmarks are identified
4. Specific gestures are mapped to mouse actions
5. Cursor is controlled in real-time
